## 🛠️ Notable Projects

### 📊 FDOT District 6 — 5-Year Program Management Report
*FDOT District 6 | 2026*

Designed and delivered the authoritative 5-year work program funding report for District 6 program managers and leadership. The solution eliminates manual data compilation from disparate systems by fully automating the reporting pipeline through SQL Server stored procedures, Power BI, and SSRS.

**Highlights:**
- Built a multi-stage SQL Server pipeline (`Create5yPM_REPORT`) that consolidates funding data across all active work program items and phases, covering a 7-year fiscal window (2026–2032)
- Implemented a three-tier version precedence system (Tentative > Adopted > Candidate) and deferred-row exclusion logic to ensure only the most current, authoritative funding data reaches decision-makers
- Delivered three reporting surfaces: an **Excel workbook** for offline planning, an **SSRS Paginated Report** hosted on Power BI Government cloud (filterable by program manager), and a **Power BI Interactive Dashboard** for trend analysis and QA validation
- Integrated data from multiple source systems (`DB2SQL1`, `PSEE`) with cross-database joins and indexed output tables optimized for manager, FPID, and phase filtering

**Stack:** SQL Server · T-SQL · Power BI · SSRS · DAX · Power BI Government Cloud
