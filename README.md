*GFT / FDOT District 6 · 2026*

Designed and delivered the authoritative 5-year work program funding report for District 6 program managers and leadership, eliminating manual data compilation from disparate systems.

- Integrated data from `DB2SQL1` and `PSEE` with cross-database joins and indexed output tables optimized for manager, FPID, and phase filtering
